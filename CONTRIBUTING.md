# Contributing Guide

Thank you for your interest in contributing to **Tri-Objective Robust XAI for Medical Imaging**.

The primary goals of this repository are:

- high quality code for an MSc dissertation,
- strong reproducibility guarantees,
- a clean foundation for future research publications.

Please read this document before opening issues or pull requests.

---

## 1. Ways to Contribute

- **Bug reports** – unexpected errors, failing commands, incorrect results.
- **Documentation** – clarifying installation steps, adding examples, improving docstrings.
- **Tests** – adding unit tests to increase coverage, especially for critical modules.
- **Features** – new loss functions, attacks, datasets, or evaluation metrics.

For substantial changes, open an issue first and describe your idea.
